# dataset-DZ5
Данные о продажах автомобилей аукционна Manheim (США), аукционн специализируется на продаже поддержанных автомобилей как физлиц так и от диллеров.
https://www.kaggle.com/datasets/syedanwarafridi/vehicle-sales-data
Набор данных «Продажи автомобилей и рыночные тенденции» представляет собой исчерпывающую коллекцию информации, касающейся сделок купли-продажи различных автомобилей. Этот набор данных включает в себя такие сведения, как год выпуска, марка, модель, комплектация, тип кузова, тип трансмиссии, VIN (идентификационный номер транспортного средства), штат регистрации, оценка состояния, показания одометра, цвет кузова и салона, информация о продавце, значения из отчета Manheim Market Report (MMR), цены продажи и даты продажи.
